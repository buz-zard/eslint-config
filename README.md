## Installation

- `npm i -D @k.sh/eslint-config`

- Update `.eslintrc` file

```json
{
  "extends": ["@k.sh/eslint-config"]
}
```

- Create a `.prettierrc` file

```json
{
  "semi": true,
  "singleQuote": true,
  "trailingComma": "es5"
}
```
